# 🌱 EcoTrack – Personal Carbon Footprint Tracker

EcoTrack is a web-based personal carbon footprint tracking application that helps users monitor their daily environmental impact by recording activities related to transportation, food consumption, and energy usage. The application calculates carbon emissions, provides interactive visualizations, and offers AI-powered eco-friendly suggestions to encourage sustainable living.

---

## 📌 Features

- 🔐 User Registration & Login
- 🚗 Track Transportation Activities
- 🍽️ Log Food Consumption
- ⚡ Monitor Household Energy Usage
- 📊 Interactive Dashboard with Charts
- 📅 View Carbon Footprint History
- 🎯 Personal Carbon Budget Tracking
- 🏆 Achievement Badges & Progress Tracking
- 🤖 AI-powered Eco-Friendly Suggestions
- 📱 Responsive User Interface

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/EcoTrack.git
cd EcoTrack
```

---

### 2. Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

Backend will run at:

```
http://127.0.0.1:5000
```

---

### 3. Frontend Setup

Open another terminal.

```bash
cd frontend
npm install
npm start
```

Frontend will run at:

```
http://localhost:3000
```
